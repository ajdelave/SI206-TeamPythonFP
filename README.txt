Desription:
This program takes personalized information from Spotify's and Apple Music's APIs to deliver data visulations.

Instructions for use: (Spotify Visualization)
1. Open Terminal or Command Line.
	- Install required packages
		- pip install spotipy
		- pip install seaborn

2. Open the spotify_info.py documents:
	- Enter API Keys, Consumer Keys, and other required information.
	- NOTE: This program requires development autorization from Spotify because it uses personal data.
		For ease of use, Andrew's personal API keys are still in the file.
	
3. Run run.py
